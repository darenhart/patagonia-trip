# Cycling Patagonia
Progressive Web App made with Quasar + Vue.js about my Patagonia bike tour in 2017-2018

## Run
```
npm run dev
```

## Deploy to git pages
----
```
npm run build
npm run deploy
OR
git checkout gh-pages
cp dist/spa-mat/* . -R
// git add modified files
git push origin gh-pages
```
